  1. Start from the root node.
  1. Compare the new value with the value of the current node.
  1. If the new value is smaller, move to the left child.
  1. If the new value is greater, move to the right child.
  1. Continue moving left/right until you reach an empty position (NULL).
  1. Insert the new value at that empty position.
  1. BST property is automatically maintained because every left child is smaller and every right child is larger.
